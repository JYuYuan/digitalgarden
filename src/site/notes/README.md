## 🛠️ 核心功能

### 1. 自动化仪表板
- Dataview 自动统计
- 按技术栈/业务系统分类
- 实时更新，无需维护

### 2. 智能搜索
- 标签搜索：`tag:#Andromeda系列`
- 全文搜索：`Cmd/Ctrl + Shift + F`
- 组合搜索：`tag:#需要升级 tag:#React-16`

### 3. 问题记录系统
- 快速记录问题和解决方案
- 自动归档（≥50 条）
- 命令行工具：`kb-add`

### 4. Claude Code 集成
- 自动识别项目
- 主动调用知识库
- 问题记录提醒

## 🎯 日常使用

### 早上开始工作
1. 打开 [[00-仪表板\|00-仪表板]]
2. 查看项目统计
3. 点击项目 → 查看快速启动

### 切换项目
```bash
cd project-name
# 可选：按 [[07-指南文档/优化工作流程指南\|优化工作流程指南]] 配置 kb-start 后显示启动信息
```

### 遇到问题
1. 搜索知识库：Obsidian 全文搜索，或按 [[07-指南文档/优化工作流程指南\|优化工作流程指南]] 配置 `kb-search`
2. 如果没找到 → 解决后记录
3. 告诉 Claude Code："记录一下：问题和方案"

# 常见问题解决方案

> 记录开发过程中遇到的常见问题和解决方案

## React 相关

### 问题 1：React 16 升级到 React 18

**适用项目**：[[02-Web端项目/tpm-pc-manager\|tpm-pc-manager]]

**问题描述**：
- 需要升级 React 16.13.1 到 React 18
- Ant Design 4 到 5 的兼容性

**解决方案**：
```bash
# 1. 升级 React
npm install react@18 react-dom@18

# 2. 更新 ReactDOM.render
# 旧写法
ReactDOM.render(<App />, document.getElementById('root'));

# 新写法
import { createRoot } from 'react-dom/client';
const root = createRoot(document.getElementById('root'));
root.render(<App />);

# 3. 升级 Ant Design
npm install antd@5
```

**注意事项**：
- React 18 的 Automatic Batching 可能影响现有代码
- Ant Design 5 有破坏性更新，需要查看迁移指南
- 第三方组件库可能需要同步升级

### 问题 2：Redux Toolkit 最佳实践

**适用项目**：[[02-Web端项目/tpm-pc-manager\|tpm-pc-manager]]

**推荐写法**：
```typescript
// 使用 createSlice
import { createSlice, PayloadAction } from '@reduxjs/toolkit';

interface UserState {
  name: string;
  age: number;
}

const userSlice = createSlice({
  name: 'user',
  initialState: { name: '', age: 0 } as UserState,
  reducers: {
    setUser: (state, action: PayloadAction<UserState>) => {
      state.name = action.payload.name;
      state.age = action.payload.age;
    }
  }
});

export const { setUser } = userSlice.actions;
export default userSlice.reducer;
```

## 小程序相关

### 问题 4：微信小程序 npm 包使用

**适用项目**：[[03-小程序项目/andromeda-wx\|andromeda-wx]]

**步骤**：
1. 在项目根目录执行 `npm install`
2. 在微信开发者工具中：工具 → 构建 npm
3. 在代码中引入：`import lib from 'package-name'`

**常见问题**：
- 构建后找不到包 → 检查 `package.json` 中的包是否在 `dependencies`
- 报错 `module not found` → 删除 `miniprogram_npm` 重新构建

## 性能优化

### 问题 5：大列表渲染优化

**适用场景**：表格数据量大的页面

**解决方案**：
```typescript
// 使用虚拟滚动
import { FixedSizeList } from 'react-window';

const VirtualList = ({ items }) => (
  <FixedSizeList
    height={600}
    itemCount={items.length}
    itemSize={50}
    width="100%"
  >
    {({ index, style }) => (
      <div style={style}>{items[index]}</div>
    )}
  </FixedSizeList>
);
```

## 构建部署

### 问题 6：Webpack 构建速度优化

**适用项目**：[[02-Web端项目/tpm-pc-manager\|tpm-pc-manager]]

**优化方案**：
```javascript
// webpack.config.js
module.exports = {
  // 使用 cache
  cache: {
    type: 'filesystem'
  },
  
  // 优化 resolve
  resolve: {
    extensions: ['.js', '.jsx', '.ts', '.tsx'],
    modules: ['node_modules']
  },
  
  // 使用 thread-loader
  module: {
    rules: [{
      test: /\.tsx?$/,
      use: ['thread-loader', 'ts-loader']
    }]
  }
};
```

**审计端移动应用共享契约：**
- [[03-小程序项目/tpm-evidence-audit-wxapp\|tpm-evidence-audit-wxapp]] 与 [[04-移动端项目/tpm-evidence-audit-android\|tpm-evidence-audit-android]] 共享 `liveFollow` 直播陪审、现场拍照、转录播、TIM 信令和 TRTC 直播流程。
- 共享接口包括 `live/app/v1/task/detail/query`、`live/app/v1/line/up`、`live/app/v1/connected/check/{taskCode}`、`live/app/v1/im/sig/create`、`live/web/v1/task/mix/start`、`live/web/v1/task/mix/stop`、`live/app/v1/video/record/save`。
- 共享 TIM key 包括 `pushUrl`、`liveStop`、`checkForm`、`msg`、`liveQueueStatus`、`fail`、`grantVideoStatus`、`getPushUrl`、`getLocation`。
- TRTC 约定：SDK App ID `1400380294`，用户 ID 为 `<当前 appUserCode>_auditUserCode<审核员 appUserCode>`，`taskCode` 同时作为房间 ID 和流 ID，混流 ID 为 `mix_<taskCode>_<审核员 appUserCode>`。
- 跨端改接口、TIM key、TRTC 参数、混流 payload 或任务状态时，需要同时检查两个移动端项目。

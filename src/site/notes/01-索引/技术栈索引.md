### TPM 审计端 TIM/TRTC 跨端契约

[[03-小程序项目/tpm-evidence-audit-wxapp\|tpm-evidence-audit-wxapp]] 和 [[04-移动端项目/tpm-evidence-audit-android\|tpm-evidence-audit-android]] 使用不同平台 SDK，但共享同一套直播审核业务契约：

| 契约项 | 小程序 | Android |
|--------|--------|---------|
| IM SDK | `tim-wx-sdk` | `imsdk-plus` |
| TRTC SDK | `trtc-wx-sdk` + 微信 `LivePusher` | `LiteAVSDK_TRTC` + `TRTCCloud` |
| SDK App ID | `1400380294` | `1400380294` |
| 房间/流 ID | `taskCode` | `taskCode` |
| 用户 ID | `<当前 appUserCode>_auditUserCode<审核员 appUserCode>` | 同左 |
| 混流 ID | `mix_<taskCode>_<审核员 appUserCode>` | 同左 |
| 关键 TIM key | `pushUrl`、`liveStop`、`checkForm`、`getLocation`、`getPushUrl` | 同左 |

改 TIM key、TRTC 参数、混流 payload 或直播审核接口时，需要同时检查两个移动端项目。
